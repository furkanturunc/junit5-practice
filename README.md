# JUnit5 Practice Project

This is a **Java project** created to practice **JUnit 5 unit testing**.

---

## Project Overview
- Purpose: Learn and practice JUnit 5 features.
- Includes: Example Java classes and corresponding unit tests.
- Focus Areas:
  - Writing unit tests
  - Using assertions
  - Test static and non-static methods 
  - Parameterized tests
  - Organizing test classes
